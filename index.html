Hi! My name is Victor Basumatary and I build web applications for a living. I learned web development through self study, and trial and error, and I want to share what I've learned over the years with everyone else out there who are also interested in learning web development on their own.

Why?

Because I want to challenge myself.

I feel most college curriculums and online tutorials/courses are lacking. They teach you fundamentals like algorithms and data structures, how to use tools like React, jQuery, databases, and maybe in the end you'll also build a simple eCommerce website or chat application as a project. I don't want to downplay what these courses and curriculums teach you (because they're all important) but I feel that they could be better. Much better.

Goal.

A free, easy, and holistic course on web development.

I have to explain myself a little here - I like to draw a line between a website and a web app. I'm not an academic type so you don't have to use my definitions of these two terms.

Website
A website is made of HTML, CSS, and JavaScript code that runs exclusively within a browser context.
Web App
A software application that runs on remote server(s) on the internet that users may or may not be able to interact with directly.

I make this distinction because there is a lot of literature out there that uses these two terms interchangeably and in my opinion it just muddies the water when you're trying to learn this stuff.

What will I learn?

Web development as I've learned it and am learning it.

A lot of developers in the web development industry are either "Frontend Engineers" or "Backend Engineers". A few are "Full Stack Developers". I believe it is more empowering for a web developer to understand and be able to implement all components of a solution. This course will NOT focus on tools and technologies as the primary concern of learning but rather the thought process and principles involved in building complete solutions. The learner is encouraged to use supplementary material in their effort to learn web development.

Who is this for?

Hopefully, everyone.

Having some prior programming experience will definitely help you but I want to try and make this course approachable for people with no programming experience as well.
